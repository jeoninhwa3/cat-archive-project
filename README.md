# 프로젝트 소개

- 프로젝트 명 : 갓생챌린지
- 소개
  - 한 줄 정리 : 도전하고 싶은 갓생챌린지를 올리고 갓생메이트를 찾는 커뮤니티
  - 내용 :
  - 갓생챌린지는 갓생러들이 원하는 챌린지를 공유하고 뉴스피드 형식으로 탐색, 참여 할 수 있는 공간입니다.
